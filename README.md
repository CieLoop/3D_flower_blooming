# Flower Collection

A collection of beautiful animated flower visualizations created with HTML5 Canvas.

## Project Contents

This project contains three different flower animations:

1. **flower1/flower.html** - Simple Flower Animation
2. **flower2/index.html** - Rose Animation  
3. **Lotus/lotus.html** - Lotus Animation

## How to Run

### Option 1: Direct File Opening (Easiest)
Simply double-click the HTML file to open it in your default web browser:
- `flower1/flower.html`
- `flower2/index.html`
- `Lotus/lotus.html`

### Option 2: Using VS Code Live Server
If you have the Live Server extension installed in VS Code:
1. Right-click on the HTML file in the file explorer
2. Select "Open with Live Server"
3. The file will automatically open in your browser

## Requirements

- A modern web browser (Chrome, Firefox, Edge, Safari, etc.)

These animations use HTML5 Canvas and should work on:
- Chrome/Chromium
- Firefox
- Safari
- Edge
- Any modern browser supporting Canvas API

## Features

Each animation includes:
- Full-screen responsive design
- Smooth canvas-based animations
- Beautiful gradient backgrounds
- Decorative text messages
- Font animations with glow effects
